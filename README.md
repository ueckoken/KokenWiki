# KokenWiki

KokenWiki is a Markdown wiki system built on top of Ruby on Rails

## Software stack

- Ruby 3.0
  - Rails 6.1
- Node.js 12
  - React + TypeScript
- MySQL 8.0

## Development

Docker and docker compose are recommended to setup your development environment

1. Start services
   - `docker compose up`
1. visit local development server localhost:3000
